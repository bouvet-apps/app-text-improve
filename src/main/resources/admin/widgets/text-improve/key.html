This widget needs an openai API key in order to communicate with ChatGPT.
Follow the steps below to create an API key and add it to the widget:

  • Go to the openai webpage
  • Click log in to create a free account
  • Select API
  • Hover over the openai icon to the left to open the usermenu
  • Select API keys and create a new key
  • Copy the key and add it to a file called no.bouvet.app.textimprove.cfg like so:
    apiKey = YOUR KEY HERE
  • Store the file on the Enonic server
    (if the server is hosed by Enonic, then they have to do this part)
